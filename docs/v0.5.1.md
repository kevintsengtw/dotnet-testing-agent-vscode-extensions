- `dotnet-testing-agent-0.5.1.vsix`

## 安裝 / 更新

本 Extension 以 `.vsix` 發佈、不走 Marketplace，**VS Code 不會自動更新**。一般更新就地覆蓋即可。

**安裝步驟**

1. 開啟 Extensions 面板（`Ctrl+Shift+X`）
2. 點選右上角 `...` → **Install from VSIX...**
3. 選擇 `dotnet-testing-agent-0.5.1.vsix`
4. **完整結束並重新開啟 VS Code**（`File → Exit` / macOS `Cmd+Q`，**不是只 Reload Window**）

**命令列**

```bash
code --install-extension dotnet-testing-agent-0.5.1.vsix
```

> ⚠️ 若從 v0.4.0（或更舊）升級而**未看到「Codex 模式」面板**：VSIX 就地覆蓋後只 Reload Window 新面板不會出現，**完整結束 VS Code 再重開即可**（不需解除安裝）。本版起，偵測到剛更新時「版本資訊」面板會主動顯示此提示。

## 主要變更

- **「剛更新」提示橫幅**：偵測到版本與上次啟動不同時，於「版本資訊」面板頂端顯示提醒——若側邊欄未看到新面板，請完整結束 VS Code 再重開（非 Reload Window）。可手動關閉，完整重啟後自動消失
- **更新指引補強**：README / 安裝說明明確標示「安裝新版後需完整重啟」
