# dotnet Testing Agent — Copilot 版 v0.1.2 安裝指南

## 前置需求

| 項目 | 最低版本 |
| --- | --- |
| VS Code | 1.96.0 |
| Node.js | 18.0.0 |
| GitHub Copilot Chat | 最新版（已登入） |
| 作業系統 | Windows 10/11（macOS / Linux 實驗性支援） |

## 下載

從 [Releases](https://github.com/kevintsengtw/dotnet-testing-agent-vscode-extensions/releases/tag/copilot%2Fv0.1.2) 下載：

- `dotnet-testing-agent-copilot-0.1.2.vsix`

## 安裝

**方法一：Extensions 面板**

1. 開啟 Extensions 面板（`Ctrl+Shift+X`）
2. 點選右上角 `...` → **Install from VSIX...**
3. 選擇下載的 `.vsix` 檔案

**方法二：命令列**

```bash
code --install-extension dotnet-testing-agent-copilot-0.1.2.vsix
```

## 初始化

安裝後執行初始化：

1. 開啟命令面板（`Ctrl+Shift+P`）
2. 執行：`dotnet-testing 測試工作流程: 初始化 .NET Testing Agent`
3. 選擇安裝模式（線上 / 離線）
4. 等待 6 個步驟完成

## 確認 MCP Server 啟用

1. 開啟 Copilot Chat（`Ctrl+Alt+I`）
2. 切換到 **Agent** 模式
3. 點選工具圖示（錘子 🔨）
4. 確認清單中有 `dotnet-testing-skills`，且 `query_documents`、`list_files` 已勾選

## 主要變更

- 新增離線安裝模式（`dta.mcpInstallMode`）：init 時可選擇線上（HuggingFace 自動下載）或離線（本地 zip）安裝 embedding model
- MCP Server 跨平台相容性修正（macOS / Linux `npx` 路徑解析）
- bundled orchestration 升級至 v2.0.0
