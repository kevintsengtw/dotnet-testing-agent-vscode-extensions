## 下載

從 [Releases](https://github.com/kevintsengtw/dotnet-testing-agent-vscode-extensions/releases/tag/v0.2.1) 下載：

- `dotnet-testing-agent-0.2.1.vsix`

## 安裝 / 更新

本 Extension 以 `.vsix` 發佈、不走 Marketplace，**VS Code 不會自動更新**。從舊版更新時不需先解除安裝，VS Code 會就地以新版覆蓋。

**方法一：Extensions 面板**

1. 開啟 Extensions 面板（`Ctrl+Shift+X`）
2. 點選右上角 `...` → **Install from VSIX...**
3. 選擇下載的 `.vsix` 檔案
4. 依提示 **Reload Window**

**方法二：命令列**

```bash
code --install-extension dotnet-testing-agent-0.2.1.vsix
```
